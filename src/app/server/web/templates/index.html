It is a service-applied web-program based on OSTIS technology, named as Ontologies management system (OMS), to automatically solve problems of loading and unloading owl-ontologies into the OSTIS knowledge base, editing them and searching for information in them.

Used technologies and means in application:

  • Python and C++ languages as main programming languages to develop its application;

  • OSTIS Technology to store, represent and edit uploaded owl-ontologies;

  • Python web-oriented json-protocol to access OSTIS memory;

  • RDF-lib for translating owl-ontologies into SC-code and vice versa;

  • FLASK and Boostrap frameworks to represent information in a RESTful web-interface;

  • Docker-compose building to automatically start servers.

The application currently allows to solve next expert tasks:

  1. Upload owl-ontologies and save their on knowledge base.

  2. Show owl-ontologies in knowledge base.

  3. Edit knowledge base ontologies using supported CRUD operations.

  4. Search object specifications in knowledge base ontologies.

  5. Download owl-ontologies from knowledge base.
